# Angular Data Table

A reusable data table component built with pure Angular

## Features

- Column sorting
- Global search
- Pagination
- Responsive (mobile, tablet, desktop)
- Custom cell rendering

## Requirements

- [Node.js](https://nodejs.org/) v18 or higher
- npm v9 or higher

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/helios58/angular-data-table.git
cd angular-data-table
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

The app will be available at `http://localhost:4200`
